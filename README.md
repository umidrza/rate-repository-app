# Rate Repository App

A React Native application for browsing GitHub repositories and posting reviews.

Scan the QR code below using Expo Go.

![EAS Update QR Code](docs/qr-code.png)

## Features

- Authentication
- Browse repositories
- Search & sort repositories
- Infinite scrolling
- View repository details
- Create reviews
- View user's reviews
- GraphQL + Apollo Client
- React Navigation

## Tech Stack

- React Native
- Expo
- Apollo Client
- GraphQL
- React Router Native
- Formik
- Yup
